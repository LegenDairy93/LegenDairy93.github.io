Projects

promptdiff

open source · TypeScript

A local-first CLI for prompt regression testing. Prompts are code — this diffs them like code. Runs test cases, validates output, stores git-friendly run artifacts, and fails CI when behaviour regresses.

  • Assertions: contains, regex, json_schema, max_length
  • Plain-JSON run artifacts with prompt hash, provider, and full assertion results
  • diff two runs — pass-rate delta, newly passing/failing cases
  • CI regression gate: non-zero exit when regressions exceed threshold
  • Deterministic offline mock provider; API keys never written to artifacts

Zaraki — natural-language research engine

private · closed beta

Built end-to-end as part of a founding team: a research engine that compiles plain-language intent into validated, executable strategy specifications, runs them safely, and tracks every result.

  • Sandboxed evaluator — whitelisted primitives, no arbitrary code execution, timeouts, memory isolation
  • Simulation pipeline producing performance metrics, with run-level lineage for reproducibility
